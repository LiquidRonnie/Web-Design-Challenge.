Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. to accomplish this, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.


After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude by analysizing four different factors: Temperature, Cloudiness, wind speed and Humidity. This site was create to provide the source data and Visualizations created, along with observations made during the analysis.
